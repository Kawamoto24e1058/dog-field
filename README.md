# Dog Field - オンライン マッチングゲーム

🐕 ブラウザで遊べるカードマッチングゲームです！

## ゲームの説明

Dog Fieldは、犬をテーマにした楽しいカードマッチングゲームです。すべてのペアを見つけて、最高記録を目指しましょう！

### 遊び方

1. 「ゲームスタート」ボタンをクリックしてゲームを開始
2. カードをクリックして裏返す
3. 同じ絵柄のカード2枚を見つける
4. すべてのペアを見つけたらクリア！

### 特徴

- 🎮 シンプルで直感的な操作
- ⏱️ タイムトラッキング機能
- 📊 移動回数カウント
- 🎨 美しいグラデーションデザイン
- 📱 レスポンシブデザイン（スマホ対応）
- 🐶 可愛い犬関連の絵文字（犬、骨、ボール、家など）

## プレイ方法

1. リポジトリをクローン
2. `index.html` をブラウザで開く

または、Webサーバーを起動：

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセス

## 技術スタック

- HTML5
- CSS3（グラデーション、アニメーション）
- Vanilla JavaScript（ES6+）

## ゲーム画面

ゲームは4x4のグリッドで、8種類のカードペア（合計16枚）をマッチさせます。

楽しんでください！ 🎉
